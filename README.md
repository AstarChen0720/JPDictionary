上帝的建築工地總結（JPDictionary 專案）
上帝（開發者）： 你是最高意志，掌管一切，負責編寫「規則」與「要求」。

網頁（房子）：

HTML： 房子的硬體設施（牆壁、屋頂、窗戶）。

CSS： 房子的軟裝與材質（顏色、壁紙、甚至是特定材質的牆）。

JavaScript： 自動觸發的規則（例如有人進廚房就開燈）。

瀏覽器（工人派遣公司）： 負責在各種環境（地形）下，按照你的要求蓋出一模一樣的房子，並將複雜的硬體轉化為 DOM（建築圖） 方便管理。

React（主管）： 因為工人太笨，上帝請來主管。上帝只需對主管下簡單的命令，主管會負責處理繁瑣的細節。

Vite（工頭/翻譯官）： 主管（React）講的話工人聽不懂，需要工頭翻譯。且工頭很聰明，會實行「局部施工」，改窗戶就只改窗戶，不拆整棟房。

Node.js（基礎設施）： 工地現場的平地、自來水與電力系統，沒有它，任何工頭或工人都無法開工。

npm（物流中心/服務商）： 所有在 Node.js 基礎上運作的東西、家具、甚至是請主管（React）或工頭（Vite）的勞務服務，都從這裡叫貨。

原型屋（Vite 預設專案）： 工頭一來就先用他帶來的工具和 React 主管，在 localhost:5173 這片工地上搭好了一個樣品屋給你測試。
